This is an Online Store for Pit's Car Accessories. Made with Node, Express, MongoDB. It also has it's own Content Management System so managers could Add, Delete, Update users and the store's products.

## User Manual

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image024.png?raw=true)

### Home Page

This page is displayed when a guest visits the website

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image025.png?raw=true)

### Register Page

A guest must register first to become a user. Once registered, the web application will send an email to the user to validate the user account.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image026.png?raw=true)

### Error Messages

If the guest tries to do something that only a user could do. The application will force the guest to log in as a user.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image027.png?raw=true)

### Store Page

Once a user logs in, the application will redirect them to the Store page. Where the user could view, go to an item’s page, and add the item they want in their shopping cart.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image028.png?raw=true)

### Enlarge View

Users could enlarge the pictures of a product in the store page.
Quick View

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image029.png?raw=true)

### Quick View

Users could also view the item on the same page using the quick view button on the Item Card.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image030.png?raw=true)

### Product Page

This page is where more information on an item can be viewed. It is also where a user can view reviews for the item as well as add his own review.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image031.png?raw=true)

### Product Page Item Review

On this part of the page, users could read the reviews of other users as well as add, edit, and delete the users own reviews. If the user already has a review on a product, the submit review button is disabled for that user.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image032.png?raw=true)

### Add-to-Cart Button

Users could add item to their cart using the Add-to-Cart Button.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image033.png?raw=true)

### Cart-Quick-View

Once a user has added an item in the cart. The user could view what’s inside their cart using the Cart-Quick-View. Users could also directly go to the checkout page using the Cart-Quick-View, go to the cart page to review their items, and remove items from their cart.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image034.png?raw=true)

### Cart Page

Users could review their cart using in the cart page. They could add and subtract the quantity of the items here as well as remove an item in the cart. Once the user has reviewed his cart, they could click the checkout button to check out or the continue-shopping button to redirect them to the store if they’re still not done shopping.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image035.png?raw=true)

### Checkout Page

In the Checkout page, the user must type in the address of where the items will be delivered.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image036.png?raw=true)

### Checkout Page Payment Method

The user must also choose the type of payment method that they want.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image037.png?raw=true)

### Checkout Page Confirm Order

Once the user has typed in the address of the delivery and chose the payment method, the user will be able to click the confirm-order button to complete the order. If the user is not done shopping yet, the user could click the cancel button to return to the cart page.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image038.png?raw=true)

### Order History Page

Once the order is complete, the user will be redirected to the user’s own order history page. Where the customer will see the order details.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image039.png?raw=true)

### User Profile Page

In this page the user could edit his account information as well as view his order history.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image040.png?raw=true)

### Edit User Account Page

In this page, the user could update his information.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image041.png?raw=true)

### Order History Page

In this page, the user could view the details of all his orders.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image042.png?raw=true)

### Admin Panel

In the Admin panel, the Admins could, see the list of products, add a product, edit a products details, delete products, see the list of customers, complete an order once they’ve sent the product for delivery, view new and old orders and go to the store page.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image043.png?raw=true)

### Product List Page

In this page, the admins could check the item’s details, edit the item’s details and as well as delete that item.

### Add Admin Navigation Button

The Super Admin could do what the Admins can do but only the Super Admin could see this navigation button. This navigation button links to the page where the Super admin could add or remove admin privileges to users and Admins can’t access this page.
Add

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image054.png?raw=true)

### Add Admin Page

In this page, The Super Admin could add and remove admin privileges to users.

![alt text](https://github.com/zenakent/Pit-s-Car-Accessories/blob/master/public/images/image055.png?raw=true)

All the Super Admin has to do to add or remove admin privileges to users is to click the Add-Admin-Button to add admin privileges or click the Remove-Admin-Button to remove admin privileges and confirm with the prompt.
